## API-Key ##

<br>

Muss gesetzt sein in Umgebungsvariable `GEMINI_API_KEY`.

Beispiel:
```
set GEMINI_API_KEY=....
```

In *AI Studio* kann über den Eintrag "Get API Key" in der Leiste links ein API-Key erzeugt werden.
